# CRUD-JavaFX
Um CRUD basico demonstrando funcionalidades do JavaFX com FXML numa arquitetura de software moderna.

- Utiliza Hibernate como ORM
- Banco de dados PostgresSQL através de um conteiner do Docker
- Migrations admnistradas através do Flyway
- Testes unitários de integração ultilizam JUnit, Stub Runner e Testcontainers
- Documentação automatizada das APIs com OpenAPI

O projeto foi desenvolvido seguindo o padrão de Conventional Commits, dentre outras boas praticas de desenvolvimento de software.
